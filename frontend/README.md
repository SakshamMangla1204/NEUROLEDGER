The frontend is now a React multi-page application built with Vite.

Run locally:

1. `cd frontend`
2. `npm install`
3. `npm run dev`

Build for backend serving:

1. `cd frontend`
2. `npm run build`
3. start the backend
4. open `http://127.0.0.1:5000`

Pages included:

- `Dashboard`
- `Identity`
- `Wearables`
- `Reports`
- `Verification`
